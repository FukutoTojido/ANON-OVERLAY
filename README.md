# ANON OVERLAY
A tosu overlay by [ANON TOKYO](https://osu.ppy.sh/users/8266808). With many features including leaderboard, key overlay graph, player profile fetching, result screen, etc. Currently only support osu! standard.

The overlay uses Vite framework with TypeScript, and should be bundled before using in production environment.

## Development
It is best to develop and test this using [Bun](https://bun.sh/). But you can use any other NodeJS runtime if you want.
```shell
bun install
bun dev
```

## Build
```
bun run build
```
